# To Do List Project

A simple to-do list application built with vanilla JavaScript to practice DOM manipulation and local storage.

## Features
- Add new tasks
- Mark tasks as complete
- Delete tasks
- Data persists using localStorage

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6+)

## Skills Practiced
- DOM manipulation
- Event handling
- localStorage API
